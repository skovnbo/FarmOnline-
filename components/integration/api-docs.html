Complete API reference for FarmOnline+ integration

Getting Started

Learn how to authenticate and make your first API calls to FarmOnline+

curl -H "Authorization: Bearer YOUR_API_KEY" \
     -H "Content-Type: application/json" \
     https://api.farmonline.com/v1/farms

Endpoints

  • GET /farms - List all farms
  • GET /farms/{id}/sensors - Get sensor data
  • POST /farms/{id}/alerts - Create alert
  • PUT /farms/{id}/settings - Update settings
