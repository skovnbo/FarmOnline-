BlueControl Connection Models

Choose the right connectivity solution for your farm's requirements. From basic offline operation to fully integrated cloud connectivity with advanced security.

Offline

Basic

BlueControl operates independently without internet connection for basic farm equipment management.

Features:

  • Configuration backup/restore via phone
  • Secure local operation
  • BlueControl companion app
  • Level 1 service agreements
Limitations:
  • No internet connectivity
  • No remote access
  • No FarmOnline+ applications
  • No OTA updates

Online

Standard

BlueControl with internet connectivity for enhanced remote support and cloud-based configuration management.

Features:

  • Internet connectivity
  • Online configuration management
  • Enhanced remote support
  • Secure connections
  • OTA updates (2026)
  • Level 2-3 service agreements
Note:
  • Limited remote access
  • FarmOnline+ requires Datalink

Online beside Datalink

Advanced

BlueControl + Datalink configuration providing full FarmOnline+ integration with comprehensive remote capabilities.

Features:

  • BlueControl + Datalink integration
  • Full FarmOnline+ Mobile access
  • Complete FarmOnline+ Web interface
  • Remote access capabilities
  • Enhanced security
  • Complete service coverage

Online behind Datalink

Enterprise

Maximum security configuration with BlueControl positioned behind Datalink for enterprise-grade protection and full functionality.

Features:

  • Maximum security configuration
  • Enhanced double security
  • Full FarmOnline+ applications
  • Complete remote management
  • Highest service level agreements
  • Enterprise network architecture
