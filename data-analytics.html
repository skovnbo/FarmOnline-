Production Analytics API

Retrieve production metrics, feed consumption, and performance indicators.

Alerts & Events API

Manage alerts, notifications, and event streams from your farm operations.

Control Systems API

Programmatically control ventilation, heating, and feeding systems.

GET /api/v1/controls/status
